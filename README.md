Este projeto faz parte de uma pipeline maior de Big Data voltada para prever o sucesso de filmes e séries. O código aqui apresentado utiliza a API do YouTube para buscar métricas de engajamento, mas ele depende diretamente dos dados previamente coletados pela API do TMDB. Esses arquivos do TMDB, que contêm informações como nome da obra, data de lançamento e popularidade, são carregados no início do script e servem como base para que o programa saiba exatamente quais títulos devem ser pesquisados no YouTube.

A partir dessa lista fornecida pelo TMDB, o script gera uma consulta padronizada combinando o nome da obra e o ano de lançamento, o que aumenta a precisão ao procurar o trailer correspondente no YouTube. Assim, as duas APIs se complementam: o TMDB fornece a identificação e os metadados das obras, enquanto o YouTube oferece indicadores de comportamento e interesse do público, como visualizações e relevância do trailer. Essa integração garante que os dados coletados formem um conjunto coerente e útil para análise posterior.
